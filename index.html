HTML as One of My Core Skills

HTML is a markup language used to structure web content.

HTML Elements Documentation

1. Structural Elements

What it is: Elements used to organize the page structure.

What it is used for: To define sections like header, main content, footer, or navigation.

When to use: Always when you want to structure your HTML properly for readability and semantic meaning.

Example:

<header>
    <h1>My Website</h1>
</header>

<nav>
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
    </ul>
</nav>

<main>
    <p>This is the main content.</p>
</main>

<footer>
    <p>© 2026 My Website</p>
</footer>

2. Text Formatting Elements

What it is: Elements used to style text semantically.

What it is used for: To emphasize, highlight, or format text.

When to use: Use them instead of styling text only with CSS.

Example:

<p>This is <strong>important</strong> text.</p>
<p>This is <em>emphasized</em> text.</p>
<

3. Lists

What it is: Ordered or unordered collections of items.

What it is used for: To display multiple items in a sequence (ordered) or bullet points (unordered).

When to use: Use when you have a group of related items.

Example:

<ul>
    <li>Apples</li>
    <li>Bananas</li>
    <li>Cherries</li>
</ul>

<ol>
    <li>Step One</li>
    <li>Step Two</li>
    <li>Step Three</li>
</ol>

4. Links

What it is: Hyperlinks to other pages or websites.

What it is used for: To navigate between pages or external sites.

When to use: Use whenever you want users to go to another page or section.

Example:

<a href="https://www.google.com" target="_blank">Visit Google</a>
<a href="#section2">Go to Section 2</a>

5. Images

What it is: Element to display images.

What it is used for: To visually enhance content or provide illustrations.

When to use: Use whenever an image is necessary to convey meaning or design.

Example:

<img src="images/profile.jpg" alt="Profile Picture" width="200">

6. Tables

What it is: Element to display data in rows and columns.

What it is used for: Present structured data clearly.

When to use: Use for data, schedules, pricing tables, etc.

Example:

<table border="1">
    <tr><th>Name</th><th>Age</th></tr>
    <tr><td>Christine</td><td>20</td></tr>
    <tr><td>John</td><td>22</td></tr>
</table>

7. Forms

What it is: A container for user input elements.

What it is used for: To collect data from users (contact, registration, feedback).

When to use: Whenever you want users to submit information.

Example:

<form action="#" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
    <button type="submit">Send</button>
</form>

8. Semantic Elements

What it is: Elements that convey meaning about their content.

What it is used for: Improves accessibility, SEO, and readability.

When to use: Always use semantic tags instead of generic <div> when possible.

Example:

<header><h1>Site Header</h1></header>
<nav><a href="#">Home</a></nav>
<main><p>Main content here.</p></main>
<article><p>Blog post content.</p></article>
<footer><p>© 2026 My Website</p></footer>
